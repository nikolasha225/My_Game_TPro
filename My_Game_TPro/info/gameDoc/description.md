# Игра "Трон"  

## Как будем выводить  
Концепция вывода делается как на смартфонах, тобишь послойно. У каждого объекта есть чтото типо своего Z-индекса, и при выводе мы начинаем отрисовывать с минимального.  
Например, когда рисуем менюшку, Z-индекс у фона = 0, а у плашек выбора = 1.
